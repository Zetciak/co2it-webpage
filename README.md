## Info
CO2it - CO2 Footprint company website

## Demo
https://zetty-co2it.vercel.app/

## Usage
Node.js: v16.14.0

Open repository folder in VSC <br/>
Install all packages
```sh
npm install
```

Run it on localhost
```sh
npm run dev
```

Open website in webbrowser <br/>
http://localhost:3000
